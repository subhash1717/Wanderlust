🌍 Wanderlust Project
📌 Overview

Wanderlust is a full-stack web application built using Node.js, Express, and MongoDB.
It is a travel listing platform where users can explore places, create listings, upload images, and share reviews.

The project follows the MVC (Model-View-Controller) architecture for clean and maintainable code.

✨ Features
🔐 User authentication (Signup / Login / Logout)
🏡 Create, edit, and delete listings
🖼️ Image upload for listings
⭐ Add and manage reviews
🔍 Browse all listings
🔒 Protected routes for secure actions
📦 MongoDB database integration
🛠️ Tech Stack
Backend: Node.js, Express.js
Frontend: HTML, CSS, JavaScript, EJS
Database: MongoDB + Mongoose
Other Tools: Express Session, Passport.js (if used), Multer

🚀 Installation & Setup
git clone https://github.com/subhash1717/Wanderlust.git
cd Wanderlust
npm install

Create a .env file:

DB_URL=your_mongodb_connection_string
SECRET=your_secret_key

Run the project:

npm start

Then open:

http://localhost:3000
